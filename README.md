# Getting Started with Boilerplate-react-typescript

## Description

- This repository is a code for react-typescript users.
- This boilerplate code has pre-commit / commit-lint settings using husky hooks

## How to use

### `start project`

```bash
npm install

npm start
```

### `Linting codes`

```bash
npm run lint

npm run lint:fix
```

### `Formatting codes`

```bash
npm run format
```
